## 🌌 Live Site

**https://kalibz-international.netlify.app**

## 📋 Features

- **Homepage** — School overview, mission, facilities
- **Gallery** — Photo gallery of school events
- **Student Portal** — Academic records, grades, timetable
- **Admin Dashboard** — Staff management, student records
- **Teacher Dashboard** — Grade management, attendance

## 🎨 Design

- **Theme:** Dark forest green (#0a1f0a → #1a4d2e)
- **Accent:** Green (#4ade80), Gold (#ffd700)
- **3D animated logo** with floating animation
- **Glass-morphism** UI elements
- **Three.js** particle background
